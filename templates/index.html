About This Project

Dataset

This classifier is trained on Stanford Dogs Dataset containing over 20,000 images across 120 dog breeds. Each image has been carefully annotated and preprocessed to ensure high-quality training data.

Model

The backend uses a TensorFlow deep learning model with transfer learning techniques. It leverages a pre-trained convolutional neural network fine-tuned for dog breed recognition with high accuracy.

Uniqueness

What sets this classifier apart is its ability to provide visual breed comparisons along with predictions, helping users understand similarities between breeds and confidence scores.
